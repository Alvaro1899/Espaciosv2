Módulo 0: Conociendo tus Datos

Exploración incial de datos

En este módulo, aprenderás a cargar y explorar tus datos en R. Estos son los primeros pasos esenciales para cualquier análisis de datos.

Para este ejemplo, pueden dascargar la información de Kaggle

Carga de Datos

La carga de datos es el primer paso en cualquier análisis. En R, existen varias funciones para importar datos desde diferentes formatos. A continuación, se describen las funciones más comunes:

  • Cargar archivos CSV

    Los archivos CSV (valores separados por comas) son uno de los formatos más comunes. Para cargarlos, puedes usar las funciones read.csv() o read_csv() del paquete 'readr'.

  • Cargar archivos de texto (read.table)

    Para archivos de texto con delimitadores personalizados (como tabulaciones o espacios), se usa read.table().

  • Cargar archivos de Excel (read_excel)

    Para archivos de Excel, se utiliza el paquete readxl. Primero, instala y carga el paquete, luego usa la función read_excel().

    • Verificación

    Después de cargar los datos, es importante verificar que se hayan importado correctamente. Puedes hacerlo imprimiendo el objeto datos o revisando sus dimensiones.

    Entre lo que podemos visualizar, está primero el número de filas y columnas (34,867 × 16). Después, vemos una fracción de la tabla con los tipos de datos que contiene. En este caso, tenemos tipos como `dbl`, `dttm` y `chr`.

    Estos son algunos de los tipos de datos que podemos encontrarnos:

    Tipo de dato Significado
    dbl Double: Números decimales (valores numéricos con punto flotante).
    dttm Datetime: Fechas y horas (almacena información de fecha y hora).
    chr Character: Texto o cadenas de caracteres.
    int Integer: Números enteros.
    fact Factor: Variables categóricas con niveles predefinidos.

    Visualización de la Estructura

    Una vez cargados los datos, es crucial entender su estructura. R ofrece varias funciones para explorar rápidamente los datos.

  • Función str()

    • La función str() (structure) proporciona un resumen compacto de la estructura del dataset, incluyendo los tipos de datos de cada columna.

  • Funciones head() y tail

    • head() muestra las primeras filas del dataset (por defecto, 6 filas). tail muestra las últimas filas del dataset.

  • Verificación de nombres de columnas

    • Puedes revisar los nombres de las columnas con la función names().

    Resumen Estadístico Rápido

    Para obtener un resumen estadístico rápido de los datos, R ofrece la función summary(). Esta función proporciona estadísticas descriptivas para cada columna, dependiendo de su tipo de dato.

    Esta función calcula estadísticas como la media, mediana, mínimo, máximo y cuartiles para columnas numéricas. Para columnas de texto o factores, muestra la frecuencia de los valores.

    Hay que destacar que el tipo de resumen dependerá del tipo de dato que obtengamos.

  • Columnas numéricas: Muestra estadísticas como la media, mediana y cuartiles.
  • Columnas de texto: Muestra la longitud y el tipo de dato.
  • Columnas de fecha/hora: Muestra el rango de fechas.

    • Lo primero que destaca en el resumen obtenido es el resultado de la variable 'index'. Es importante recordar que esta variable es un índice o identificador único para cada registro en nuestra base de datos, por lo que no tiene sentido obtener un resumen estadístico de ella (como la media, mínimo o máximo). Por el momento, podemos ignorar esta variable en el análisis estadístico. Sin embargo, si deseamos asegurarnos de que en futuras ocasiones esta variable se maneje correctamente, podemos convertirla al formato de texto 'chr') al cargar los datos o después de haberlos importado.

  • Al cargar datos

    • El paquete 'readr' permite especificar el tipo de dato de cada columna al cargar el archivo. Para forzar que una columna sea leída como carácter, usa col_types y define la columna como 'col_character()'.

    En 'read.csv()', si la columna contiene números, R podría convertirlos automáticamente a tipo numérico. Para evitarlo, puedes usar el argumento colClasses para especificar el tipo de dato de cada columna.

    Si estás cargando un archivo de Excel con read_excel() del paquete 'readxl', puedes usar el argumento col_types para especificar que una columna sea leída como texto.

  • Convertir una columna a carácter después de cargar los datos

    • Si ya has cargado los datos y te das cuenta de que la columna de índice se ha leído como numérica, puedes convertirla a carácter usando la función as.character().
